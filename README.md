# GithubRepositoryViewer

## 全体の構成

### フォルダ構成

```
|-GithubRepositoryViewer
  |
  |-View <-- 画面表示のための View と View を表示するのに必要な PresentationModel を入れている
  |-Models <-- View か ViewModel に属さないクラス群を入れている
  | |-Repositories
  | |-Image
  | |-APIs
  | |-Entites
  | |-Serializations
  |
  |-ViewModels
```
